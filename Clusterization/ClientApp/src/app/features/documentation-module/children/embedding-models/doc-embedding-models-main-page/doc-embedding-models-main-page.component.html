Основне

Для найбільшої гнучкості користувач має змогу вибирати модель векторної візуалізації тексту. Кожна модель має свої якості, ціну квот та розмір вихідних ембедингів

На даний момент інтегровано три моделі: text-embedding-ada-002, text-embedding-3-small та text-embedding-3-large

Характеристики моделів

text-embedding-ada-002 text-embedding-3-small text-embedding-3-large
Вихідний вимір ембединга 1536 1536 3072
Кількість використаних квот за один елемент 5 1 8
Максимальна кількість вхідних знаків 4000 4000 4000
