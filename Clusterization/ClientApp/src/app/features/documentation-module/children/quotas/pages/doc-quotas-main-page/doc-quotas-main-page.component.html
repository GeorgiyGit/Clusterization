Основне

Система квот впроваджена задля забезпечення безпеки додатка від перевантаження за рахунок завантаження та обробки безкінечної кількості даних. У кожного зареєстрованого користувача (який підтвердив свою електронну пошту) є певні обмеження щодо користування деякими функціями сайта. При перевищенні ліміту користувач не зможе користуватися відповідною функцією, але додаткові квоти можуть бути отримані від адміністрації. Також в розробці система купівлі квот

Перша пакет квот видається одноразово після проходження верифікації пошти, кожна дія (за яку забираються квоти) коштує різну їх кількість. Також, існують різні типи квот та відповідна доступна кількість для кожного типу. Навіть якщо користувач використав всі квоти одного типу, на інші дії з додатком, не пов'язані з цим типом квот, це ніякою мірою не вплине. При ініціалізації дії яка потребує квоти, користувач в більшості випадків буде проінформований про можливу кількість використаних квот. Але не всі дії напряму відображають попередження про кількість квот, тому рекомендується ознайомитися із відповідним розділом по ціні квот на кожну операцію
