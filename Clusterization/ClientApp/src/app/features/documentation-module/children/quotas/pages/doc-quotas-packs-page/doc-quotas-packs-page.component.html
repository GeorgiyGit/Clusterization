Паки квот

Надавання квот користувачу відбувається не на пряму, а за допомогою паків квот. Кожен пак містить певну кількість квот кожного типу, які відповідно зараховуються користувачу при активації пака

Перший пак надається при верифікації електронної пошти. Також, користувач може подивитися список всіх активованих паків на відповідній сторінці аканту
