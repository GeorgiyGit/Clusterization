K-means

K-means це один із найпростіших алгоритмів кластеризації. Він працює дуже швидко, тому може використовуватися для великих масивів даних.

Параметри для створення k-means:

  • Кількість кластерів: кількість кластерів, які будуть створені при кластеризації
  • Зерно (Seed): для того щоб однакові алгоритми давали різні вихідні дані
