DBSCAN

DBSCAN це один із найефективніших алгоритмів кластеризації. Він стійкий від шумів та добре працює із заданими даними. Саме цей алгоритм найбільше рекомендується для використання. Рекомендується використовувати в двох вимірах із параметром Epsilon рівним 50%

Параметри для створення DBSCAN

  • Epsilon: визначає радіус біля точки, представлений не у абсолютному значені, а у відсотковому, причому 50% - це оптимальне значення розраховано алгоритмом
  • Мінімальна кількість точок у кластері: мінімальна кількість точок в одному кластері, ціле число
