Основне

Карта точок відповідає за відображення кінцевих кластерів. В кожному профілі своя карта точок, причому вона стає доступна після першої кластеризації профілю. Кожна точка відповідає за свій текст (коментар або текст із зовнішніх даних). Точки одного кольору відносяться до одного кластеру. Завантаження точок відбувається динамічно.

Натиснувши на конкретну точку, висвітлюється її зміст.

Зображення результату кластеризації.
