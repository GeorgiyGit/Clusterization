Завантаження каналу за id

У кожного каналу Telegram є username. Саме за ним відбувається завантаження. Для початку потрібно ввести username у відповідне поле та натиснути кнопку "додати". Якщо канал з таким username існує, він буде завантажений та доданий до списку каналів. Після успішного завантаження з'явиться відповідне повідомлення
