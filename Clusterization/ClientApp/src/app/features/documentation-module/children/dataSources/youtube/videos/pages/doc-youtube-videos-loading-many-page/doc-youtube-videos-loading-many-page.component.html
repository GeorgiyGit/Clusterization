Завантаження багатьох відео

Завантаження одного або багатьох відео відбувається на відповідній секції відкрити яку можна в додаткових функціях взаємодії каналу.

Зображення як відкрити сторінку завантаження багатьох відео

На сторінці для завантаження необхідно вибрати кілька параметрів:

  • Дата від: дата від якої завантажуються відео. Не є обов'язковим
  • Дата до: дата до якої завантажуються відео. Не є обов'язковим
  • Кількість відео: максимальна кількість відео, які можуть бути завантаженні. Є обов'язковим
  • Мінімальна кількість коментарів: мінімальна кількість коментарів у відео, щоб воно було завантажене. Не є обов'язковим
  • Мінімальна кількість переглядів: мінімальна кількість переглядів у відео, щоб воно було завантажене. Не є обов'язковим

Після цього користувач повинен натиснути відповідну кнопку завантаження та натиснути кнопку для завантаження. Це створить спеціальну фонову задачу, та відео по даним критеріям будуть завантажені
