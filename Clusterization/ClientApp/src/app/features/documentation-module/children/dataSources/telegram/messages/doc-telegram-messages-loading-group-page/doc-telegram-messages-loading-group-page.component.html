Завантаження багатьох повідомлень

Завантаження одного або багатьох повідомлень відбувається на відповідній секції відкрити яку можна в додаткових функціях взаємодії каналу

Зображення як відкрити сторінку завантаження багатьох повідомлень

На сторінці для завантаження необхідно вибрати кілька параметрів:

  • Дата до: дата до якої завантажуються повідомленя. Не є обов'язковим
  • Кількість повідомлень: максимальна кількість повідомлень, які можуть бути завантаженні. Є обов'язковим

Після цього користувач повинен натиснути відповідну кнопку завантаження та натиснути кнопку для завантаження. Це створить спеціальну фонову задачу, та повідомленя по даним критеріям будуть завантажені
