Зовнішні дані

Зовнішні дані це текстові дані подібні до коментарів з платформи Youtube. Вони об'єднані в пакет зовнішіх даних при завантажені та містять у собі текст, який використовується при кластеризації. Також можуть містити у собі ExternalId, тобто Id який вказується користувачем. Детальніше про завантаження описано в пакетах зовнішніх даних
