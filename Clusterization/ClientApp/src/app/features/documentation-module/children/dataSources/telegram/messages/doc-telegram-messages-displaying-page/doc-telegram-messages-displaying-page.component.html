Відображення повідомлення

Картка повідомлення у списку

При перегляді повідомлення у списку кожен із них відображається у вигляді картки. На ній зображено: текст повідомлення, кількість ереглядів, завантажених відповідей в систему та інші дані. Також відображається дата завантаження відео. Більшість даних можна копіювати натиснувши на них. Якщо відео вже завантажено в систему, колір його назви червоний. У іншому випадку сірий

Детальна сторінка повідомленя

Натиснувши на картку повідомленя можна попасти на основну сторінку. На ній зображається основна інформація про повідомленя та додаткові функції взаємодії

Головний список повідомленнь

Можливий перегляд усіх повідомленнь завантажених у систему, а не тільки тих, що належать до конкретного каналу. Користувач може сортувати повідомлення та шукати їх по назву. Реалізовано пагінацію, тому процес завантаження досить швидкий
