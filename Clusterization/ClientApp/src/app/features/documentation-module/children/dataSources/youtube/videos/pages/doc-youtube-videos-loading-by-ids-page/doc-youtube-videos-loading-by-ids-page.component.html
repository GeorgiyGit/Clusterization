Завантаження кількох відео

Завантаження одного або кількох відео відбувається на відповідній секції яка знаходиться на детальній сторінці каналу

У секції завантажується деяка кількість відео з відповідного каналу, які відображаються користувачеві. Можливий вибір, за яким критерієм вони відсортовані: "Спочатку новіші" або "Спочатку популярніші". Також можливий пошук за назвою. Якщо кількість завантажених відео недостатня, можливе більше їх завантаження у самому низу списку відео, натиснувши на відповідну кнопку

Після цього користувач може вибрати одне або кілька відео (назва вже завантажених відео підсвічується червоним, та їх вибрати неможливо). Та натиснути кнопку для завантаження. Це створить спеціальну фонову задачу, та відповідні відео будуть завантажені
