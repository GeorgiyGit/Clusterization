Завантаження відповідей з каналу

Кнопка для відкривання сторінки завантаження відповідей з каналу знаходиться у додаткових функціях каналу

Зображення як відкрити сторінку завантаження відповідей з каналу

На сторінці для завантаження необхідно вибрати кілька параметрів:

  • Дата від: дата від якої завантажуються відповіді. Не є обов'язковим
  • Дата до: дата до якої завантажуються відповіді. Не є обов'язковим
  • Максимальна кількість: максимальна кількість відповідей, які можуть бути завантаженні. Є обов'язковим
  • Максимальна кількість в одному повідомлені: максимальна кількість відповідей, які можуть бути завантаженні з одного повідомлення. Є обов'язковим

Після цього користувач повинен натиснути відповідну кнопку завантаження. Це створить спеціальну фонову задачу, та відповіді по даним критеріям будуть завантажені
