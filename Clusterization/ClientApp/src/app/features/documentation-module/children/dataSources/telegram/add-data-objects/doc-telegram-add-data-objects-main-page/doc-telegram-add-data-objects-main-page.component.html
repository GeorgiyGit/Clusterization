Додавання даних

Тип "Telegram Replies"

Після завантаження відповідей з платформи Telegram, стає можливим додавання їх у робочий простір. Для початку його необхідно вибрати, натиснувши відповідну кнопку в додатковому функціоналі робочого простору на основній його сторінці. При цьому вибраний робочий простір у списку буде підсвічуватися червоним кольором. Додавання можливе у кілька етапів

Для додавання відповідей існує два підходи: додавати їх зі всіх повідомлень через канал, чи вибравши одне або кілька. Всі додавання реалізовані через головну сторінку каналу в додатковому функціоналі

Додавання відповідей із всього каналу

Для додавання відповідей зі всього каналу необхідно вибрати відповідну функцію в додатковому функціоналі каналу. Після цього відкривається сторінка додавання відповідей

На сторінці необхідно вибрати кілька параметрів:

  • Дата від: дата від якої додаються відповіді. Не є обов'язковим
  • Дата до: дата до якої додаються коментарі. Не є обов'язковим
  • Максимальна кількість: максимальна кількість відповідей, які можуть бути додані. Є обов'язковим
  • Вибір між двох параметрів: додати за датою повідомлення або додати за датою відповідей. Це визначає по якій з дат будуть додані відповіді

Після натискання відповідної кнопки створиться фонове завдання та коментарі із відповідними параметрами будуть додані до вибраного робочого простору.

Додавання відповідей із певних повідомлень

Для додавання відповідей із певних повідомлень каналу необхідно вибрати відповідну функцію в додатковому функціоналі каналу. Після цього відкриється сторінка додавання відповідей

На сторінці необхідно вибрати кілька параметрів:

  • Дата від: дата від якої додаються відповіді. Не є обов'язковим
  • Дата до: дата до якої додаються відповіді. Не є обов'язковим
  • Максимальна кількість (у повідомленні): максимальна кількість відповідей, які можуть бути додані з одного повідомлення. Є обов'язковим
  • Також, необхідно вибрати повідомлення із списку, які будуть завантажені

Після натискання відповідної кнопки створиться фонове завдання та відповіді з відповідними параметрами будуть додані до вибраного робочого простору
