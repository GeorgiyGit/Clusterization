Завантаження каналів за назвою

Для початку необхідно ввести назву. Після натискання enter або кнопки пошуку відбувається пошук певної кількості каналів використовуючи вбудовані інструменти платформи Telegram. Після вибору каналів необхідно натиснути кнопку завантаження та вибрані канали завантажаться
