Основне

Для завантаження каналів необхідно авторизуватися. Всі авторизовані користувачі з підтвердженою поштою можуть завантажувати канали. На сервер завантажується детальна інформація про канал, включаючи кількість учасників, username та інша статистика

Сторінку завантаження каналів можна відкрити у Telegram header сайту. Після цього доступно два види завантаження: за username каналу або після пошуку за назвою
