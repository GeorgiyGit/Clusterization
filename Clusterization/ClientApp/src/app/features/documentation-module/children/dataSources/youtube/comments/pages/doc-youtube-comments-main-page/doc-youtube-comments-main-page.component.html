Основне

Саме коментарі з платформи Youtube є одним із джерел текстових даних для кластеризації. Їх завантаження можливе у два способи: з каналу та з конкретного відео. Платформа Youtube дозволяє завантажувати не всі опубліковані коментарі (не можна завантажити видаленні або заблоковані). З іншого боку, у статистиці відео відображається кількість усіх коментарів, включаючи видалені та заблоковані. Це створює невелику проблему відображення, коли на перший погляд на сервер завантажені не всі коментарі, хоча на справді було завантажено всі доступні. При плануванні завантаження коментарів необхідно враховувати цей факт та в деяких випадках вибирати дещо більші числа при завантажені відео.
