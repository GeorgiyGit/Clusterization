Картка каналу у списку

При перегляді каналів у списку кожен із них відображається у вигляді картки. На ній зображено: назва каналу, зображення, кількість учасників, username й кількість завантажених повідомлень у систему сайту. Також відображається дата завантаження каналу. Більшість даних можна копіювати натиснувши на них. Якщо канал вже завантажено в систему, колір його назви червоний. У іншому випадку сірий

Детальна сторінка каналу

Натиснувши на картку каналу можна попасти на основну сторінку. На ній зображається основна інформація про канал. Також розміщуються інші сторінки, такі як список завантажених повідомлень каналу та можливість завантажити більше відео. Крім цього, існують додаткові функції взаємодії
