Створення

Функція створення профілю знаходиться в додатковому функціоналі робочого простору. Додати профіль можна тільки до того робочого простору, зміна якого можлива для всіх користувачів (або ви є власником робочого простору).

Зображення, як відкрити сторінку додавання профілю

Параметри для створення One cluster

  • Тип алгоритму: тип алгоритму кластеризації. Параметр є обов'язковим.
  • алгоритм: певний алгоритм вибраного типу, який буде використовуватися під час кластеризації. Параметр є обов'язковим.
  • Модель векторної візуалізації: модель, яка буде використовуватися при завантажені ембедингів.
  • Кількість вимірів: кількість вимірів у яких буде відбуватися кластеризація (вихідний результат в будь якому разі буде проектуватися в двовимірний простір). Параметр є обов'язковим.
  • Алгоритм проекції: тип алгоритму проекції даних (зменшення розмірності). За допомогою якого алгоритму буде відбуватись відображення кінцевого результату кластеризації. У випадку вибраного виміру кластеризації не рівному 1536, кластеризація також буде відбуватися у спроектованому просторі даним алгоритмом. Параметр є обов'язковим.
  • Тип алгоритму: тип алгоритму кластеризації. Параметр є обов'язковим.
  • Область видимості: визначає, хто буде бачити профіль: всі, чи тільки його власник.
  • Область модифікації: визначає, хто може змінювати профіль: всі, чи тільки його власник.
