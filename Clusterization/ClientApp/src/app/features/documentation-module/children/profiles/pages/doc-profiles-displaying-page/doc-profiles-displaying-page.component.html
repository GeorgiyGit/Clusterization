Картка профілю у списку

У списку на сторінці робочого простору відображаються картка профілю. У ній знаходиться основна інформація про профіль, включаючи вибраний алгоритм кластеризації, кількість вимірів, алгоритм зменшення розмірності, область видимості та область модифікації.

Сторінка профілю

На сторінці профілю знаходиться вся основна інформація про нього. Також, користувач може переглядати результати кластеризації (детальніше у розділі візуалізації кластеризації та карти точок). Також, у додатковому функціоналі знаходиться кнопка кластеризації.
