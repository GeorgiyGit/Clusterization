Динамічне підвантаження

Для кращої продуктивності реалізовано динамічне підвантаження точок на мапу. Кожна мапа точок складається з рівнів, які в свою чергу діляться на квадрати. Даний підхід використовують онлайн карти (наприклад Google maps). Для неієрархічної кластеризації створюється лише один рівень, який ділиться 16 на 16, створюючи 256 рівних квадратів. При перегляді користувачем звантажуються тільки ті квадрати, які він бачить. Тобто необхідності завантажувати всі точки немає, що підвищує продуктивність при великій кількості даних.
