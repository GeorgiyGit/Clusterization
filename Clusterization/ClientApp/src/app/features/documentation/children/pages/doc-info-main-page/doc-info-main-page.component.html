Основна інформація

Цей додаток створений для автоматизації кластеризації текстових даних. Під цим розуміється повна автоматизація, включаючи підготовчі процеси та завантаження текстових даних для кластеризації. Користувач завантажує текстові дані (наприклад коментарі з платформи ютуб, причому це відбувається в не покидаючи цей сайт) та проводить гнучку кластеризацію із багатьма параметрами.

Кластерний аналіз — задача розбиття заданої вибірки об'єктів (ситуацій) на підмножини, які називаються кластерами, так, щоб кожен кластер складався з схожих об'єктів, а об'єкти різних кластерів істотно відрізнялися. Завдання кластеризації належить до статистичної обробки, а також до широкого класу завдань некерованого навчання. У випадку коментарів з платформи Youtube, кластерний аналіз допомагає групувати об'єкти фактично за сенсом.

Приклад кластеризації

Текстову кластеризацію можна умовно розділити на чотири етапи: завантаження текстових даних, створення ембедингів (або векторна кластеризація тексту), зменшення розмірності даних та відповідно кластеризація.

У розробленому веб-додатку завантаження текстових даних може відбувалися в два підходи: завантаження коментарів з платформи Youtube (використовуючи можливості веб-додатка) або завантаження зовнішніх джерел у спеціальному форматі.

Завантаження ебмедингів або векторна візуалізація тексту — це процес перетворення тексту в масив чисел (вектор). Додаток взаємодіє з моделю text-embedding-ada-002 від компанії OpenAi.

Для нормального відображення вихідних кластерів, необхідно зменшити розмірність масиву чисел. Це робиться за допомогою різних алгоритмів зменшення розмірності.

Під час кластеризації використовуються чотири основних типи алгоритмів: DBSCAN, K-means, Spectral Clustering та Gaussian Mixture. Вибравши тип користувач сам створює конкретний алгоритм із певними властивостями, тому кількість реальних реалізацій алгоритмів фактично безмежна.

Детальніше всі етапи та тонкощі описані у відповідних розділах.
