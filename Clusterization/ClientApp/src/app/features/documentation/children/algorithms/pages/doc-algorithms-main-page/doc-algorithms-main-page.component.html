Основне

Розроблений веб-додаток дає можливість гнучкого використання та вибору алгоритмів кластеризації. Реалізовано чотири основних типи алгоритмів кластеризації: k-means, DBSCAN, Gaussian Mixture та Spectral Clustering. Також було імплементовано ще один додатковий алгоритм One Cluster, хоча його не можна назвати повноцінним алгоритмом кластеризації. Це є лише типами алгоритмів, а не конкретними реалізаціями. Користувач сам створює конкретну реалізацію алгоритму із конкретними властивостями індивідуальними для кожного алгоритму.

Демонстрація роботи на прикладі алгоритму k-means. Користувачу спочатку необхідно створити конкретний алгоритм. Для цього він відкриває сторінку додавання алгоритмів, вибирає k-means. В даному випадку k-means має дві властивості: кількість кластерів та зерно. Користувач вводить необхідні йому властивості та створює алгоритм. Після цього, можливе використання алгоритму при кластеризації. Використовується саме цей алгоритм із конкретними властивостями вказаними користувачем. Така система дозволяє створювати алгоритми для конкретних випадків кластеризації, що покращує кінцевий результат.
