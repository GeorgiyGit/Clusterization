Gaussian Mixture

Gaussian Mixture це складний імовірнісний алгоритм кластеризації. Не рекомендується використовувати з виміром більшим двох через високі затрати кластеризації. При використанні з більшим виміром кластеризація буде проводитися набагато довше.

Параметри для створення Gaussian Mixture

  • Кількість кластерів: кількість створених кластерів
