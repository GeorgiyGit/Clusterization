Основне

Для завантаження відео необхідно авторизуватися. Всі авторизовані користувачі можуть завантажувати канали. На сервер завантажується детальна інформація про канал, включаючи кількість коментарів (на платформі Youtube), переглядів та інша статистика. Завантажується тільки інформація та статистика відео, завантажувати відеоряд немає необхідності.

Існує два види завантаження каналів: через вибір одного або кількох відео, або автоматичне завантаження певної кількості відео із часової межі.
