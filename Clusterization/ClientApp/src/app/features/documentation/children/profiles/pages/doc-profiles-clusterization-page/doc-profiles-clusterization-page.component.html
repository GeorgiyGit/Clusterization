Кластеризація

У профілі відбувається кластеризація із певними властивостями вибраними при створені профілю. Кластеризація можлива тільки тоді, коли всі дані робочого простору мають завантажені ембедингі.

Зображення, як почати кластеризацію

Після цього створиться фонове завдання із кластеризацією даних. В багатьох випадках це досить довгий процес, тому кластеризація може тривати дуже довго. Рекомендується використовувати алгоритм кластеризації DBSCAN, який працює досить швидко.
