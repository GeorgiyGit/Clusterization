Основне

Система профілів необхідна для ізоляції процесу кластеризації із певними властивостями. Кожен профіль міститься у якомусь робочому просторі та самостійно існувати не може.

При створенні профілю користувач вказує певні властивості кластеризації, які є не змінними під час користування профілем. Це дає змогу в одному робочому просторі проводити кілька різних кластеризацій та легко порівнювати їх результат.
