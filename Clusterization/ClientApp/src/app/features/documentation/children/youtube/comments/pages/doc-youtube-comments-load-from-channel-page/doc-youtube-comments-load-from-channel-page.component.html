Завантаження коментарів із каналу

Кнопка для відкривання сторінки завантаження коментарів із каналу знаходиться у додаткових функціях каналу.

Зображення як відкрити сторінку завантаження коментарів із каналу

На сторінці для завантаження необхідно вибрати кілька параметрів:

  • Дата від: дата від якої завантажуються коментарі. Не є обов'язковим
  • Дата до: дата до якої завантажуються коментарі. Не є обов'язковим
  • Максимальна кількість: максимальна кількість коментарів, які можуть бути завантаженні. Є обов'язковим
  • Максимальна кількість в одному відео: максимальна кількість коментарів, які можуть бути завантаженні з одного відео. Є обов'язковим
  • Вибір між двох параметрів: завантажити за датою відео або завантажити за датою коментарів. Це визначає по якій з дат будуть завантажуватися коментарі

Після цього користувач повинен натиснути відповідну кнопку завантаження та натиснути кнопку для завантаження. Це створить спеціальну фонову задачу, та коментарі по даним критеріям будуть завантажені
