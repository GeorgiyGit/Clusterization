Завантаження каналу за id

У кожного каналу Youtube є ідентифікатор (id). Саме за ним відбувається завантаження. Для початку потрібно ввести код у відповідне поле та натиснути кнопку "додати". Якщо канал з таким id існує, він буде завантажений та доданий до списку каналів. Після успішного завантаження з'явиться відповідне повідомлення
