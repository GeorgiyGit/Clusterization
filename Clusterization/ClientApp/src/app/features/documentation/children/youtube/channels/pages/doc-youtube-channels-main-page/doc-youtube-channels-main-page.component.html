Основне

Для завантаження каналів необхідно авторизуватися. Всі авторизовані користувачі можуть завантажувати канали. На сервер завантажується детальна інформація про канал, включаючи кількість відео (на платформі Youtube), підписників та інша статистика

Сторінку завантаження каналів можна відкрити у головному header сайту. Після цього доступно два види завантаження: за id каналу або після гнучкого пошуку за назвою
