Основне

Основна мета робочих просторів: ізолювати дані та роботу з ними, для зручнішої багаторівневої кластеризації. Кожен робочий простір містить у собі якісь текстові дані, які користувач може додавати до нього. Також, у ньому міститься масив профілів: об'єктів, у яких проводиться кластеризація із певними властивостями. Робочі профілі незалежні один від одного та не взаємодіють між собою.

Кожен робочий простір має деякі властивості, такі як: тип вхідних даних (коментарі з платформи Youtube або зовнішні дані у певному форматі), область видимості (яка визначає, хто може бачити робочий профіль) та область змінювання (яка визначає, хто може змінювати робочий простір).
