Завантаження ембедингів

Завантаження ембедингів чи векторна візуалізація тексту: це процес перетворення тексту в масив чисел. Він необхідний для подальшої кластеризації тексту. Детальніше про векторну візуалізацію тексту описану на відповідній сторінці.

Щоб продовжити кластеризацію, необхідно забезпечити векторну візуалізацію тексту в робочому просторі. Для цього в додатковому функціоналі необхідно натиснути на відповідну функцію. Векторна візуалізація це досить довгий процес, тому створюється фонове завдання, прогрес якого можна побачити на відповідній сторінці. Після успішного проведення векторної візуалізації тексту, у статус відповідного робочого простору зміниться, та у його профілях можна буде проводити кластеризацію.

Зображення, як почати векторну візуалізацію тексту в робочому просторі.
