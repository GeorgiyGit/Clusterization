Додавання даних

Додавання даних залежить від вибраного типу робочого простору. Якщо вибрано тип "Коментарі", додаються коментарі з платформи Youtube. При вибраному типі "З файлу", додавання коментарів реалізовано з файлу в спеціальному форматі.

Тип "Коментарі"

При вибраному типі "Коментарі" додавання даних відбувається з платформи Youtube (детальніше описано у відповідному розділі).

Після завантаження коментарів з платформи Youtube, стає можливим додавання їх у робочий простір. Для початку його необхідно вибрати, натиснувши відповідну кнопку в додатковому функціоналі робочого простору на основній його сторінці. При цьому вибраний робочий простір у списку буде підсвічуватися червоним кольором. Додавання можливе у кілька етапів.

Для додавання коментарів існує два підходи: додавати їх зі всіх відео через канал, чи вибравши одне або кілька. Всі додавання реалізовані через головну сторінку каналу в додатковому функціоналі.

Додавання коментарів із всього каналу

Для додавання коментарів зі всього каналу необхідно вибрати відповідну функцію в додатковому функціоналі каналу. Після цього відкриється сторінка додавання коментарів.

На сторінці необхідно вибрати кілька параметрів:

  • Дата від: дата від якої додаються коментарі. Не є обов'язковим
  • Дата до: дата до якої додаються коментарі. Не є обов'язковим
  • Максимальна кількість: максимальна кількість коментарів, які можуть бути додані. Є обов'язковим
  • Вибір між двох параметрів: додати за датою відео або додати за датою коментарів. Це визначає по якій з дат будуть додані коментарі

Після натискання відповідної кнопки створиться фонове завдання та коментарі із відповідними параметрами будуть додані до вибраного робочого простору.

Додавання коментарів із певних відео

Для додавання коментарів зі певних відео каналу необхідно вибрати відповідну функцію в додатковому функціоналі каналу. Після цього відкриється сторінка додавання коментарів.

На сторінці необхідно вибрати кілька параметрів:

  • Дата від: дата від якої додаються коментарі. Не є обов'язковим
  • Дата до: дата до якої додаються коментарі. Не є обов'язковим
  • Максимальна кількість (у відео): максимальна кількість коментарів, які можуть бути додані з одного відер. Є обов'язковим
  • Також, необхідно вибрати відео із списку, які будуть завантажені

Після натискання відповідної кнопки створиться фонове завдання та коментарі із відповідними параметрами будуть додані до вибраного робочого простору.

Тип "З файлу"

Також можливе завантаження текстових даних із файлу в форматі JSON, приклад наведений нижче.

Приклад файлу

Змінна "Session" у файлі дає змогу копіювати текстові дані з однаковими Id але іншими сесіями, що полегшує процес завантаження кількох файлів.

Змінна "ExternalObjects" визначає масив текстових даних, які будуть додані до робочого простору.

Щоб додати текст із зовнішніх джерел, необхідно спочатку відкрити відповідну сторінку користуючись додатковим функціоналом робочого простору.
